# 🌤️ Weather Forecast Web App

## 📖 ภาพรวมโปรเจกต์ (Project Overview)
แอปพลิเคชันพยากรณ์อากาศบนเว็บที่ใช้งานง่าย พัฒนาด้วยภาษา Python (Flask) สำหรับฝั่งเซิร์ฟเวอร์ และจัดรูปแบบหน้าตาด้วย Tailwind CSS เพื่อให้รองรับการแสดงผลทุกขนาดหน้าจอ โปรเจกต์นี้ดึงข้อมูลสภาพอากาศแบบเรียลไทม์ผ่าน Open-Meteo API โดยผู้ใช้สามารถค้นหาสภาพอากาศปัจจุบันของเมืองต่างๆ ทั่วโลกได้อย่างรวดเร็ว

## ✨ ฟีเจอร์เด่น (Key Features)
* **ค้นหาสภาพอากาศตามชื่อเมือง:** รองรับการค้นหาชื่อเมืองเป็นภาษาอังกฤษ 
* **แสดงข้อมูลสภาพอากาศครบถ้วน:** * อุณหภูมิ (แปลงเป็นหน่วยองศาเซลเซียส พร้อมปัดเศษ 2 ตำแหน่ง)
  * สภาพอากาศโดยรวม (เช่น ท้องฟ้าแจ่มใส, มีเมฆมาก/ฝนตก)
  * ความชื้นสัมพัทธ์ (%)
  * ความเร็วลม (km/h)
  * ปริมาณน้ำฝน (mm)
* **ระบบจัดการข้อผิดพลาด (Robust Error Handling):**
  * ดักจับและแจ้งเตือนกรณีผู้ใช้กรอกข้อมูลว่างเปล่า หรือกรอกเป็นตัวเลขล้วน
  * แจ้งเตือนเมื่อไม่พบชื่อเมืองที่ระบุในฐานข้อมูล
  * จัดการข้อผิดพลาดกรณีเครือข่ายขัดข้อง หรือ API ไม่ตอบสนอง (Timeout)
* **UI/UX ที่สวยงาม (Responsive Design):** หน้าเว็บออกแบบเป็นรูปแบบการ์ดที่อ่านง่าย สบายตา และรองรับการใช้งานทั้งบนคอมพิวเตอร์และสมาร์ทโฟน

## ⚙️ คำแนะนำการติดตั้ง (Installation Instructions)
ทำตามขั้นตอนด้านล่างเพื่อติดตั้งและรันโปรเจกต์นี้บนเครื่องคอมพิวเตอร์ของคุณ:

1. **โคลน Repository:**
   ```bash
   git clone [https://github.com/guzzle999/weather_app.git]
   cd weather_app
   ```

2. **สร้างและเปิดใช้งาน Virtual Environment:**
   * สำหรับ Windows:
     ```bash
     python -m venv venv
     .\venv\Scripts\activate
     ```
   * สำหรับ macOS/Linux:
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

3. **ติดตั้งไลบรารีที่จำเป็น:**
   ```bash
   pip install flask requests
   ```
   *(หรือ `pip install -r requirements.txt` หากมีการสร้างไฟล์ไว้)*

## 🚀 คู่มือการใช้งาน (Usage Guide)
1. ตรวจสอบให้แน่ใจว่าคุณอยู่ในโฟลเดอร์โปรเจกต์และเปิดใช้งาน Virtual Environment แล้ว
2. สั่งรันเซิร์ฟเวอร์ด้วยคำสั่ง:
   ```bash
   python app.py
   ```
3. เปิดเว็บเบราว์เซอร์และเข้าไปที่ URL: `http://127.0.0.1:5000`
4. พิมพ์ชื่อเมืองที่ต้องการค้นหาในช่องกรอกข้อมูล (เช่น `Bangkok`, `Tokyo`, `London`) แล้วกดปุ่ม "ค้นหา"

## 📊 ตัวอย่างผลลัพธ์ (Example Output)
เมื่อผู้ใช้ค้นหาคำว่า `"Bangkok"` ระบบจะแสดงการ์ดสภาพอากาศที่มีรายละเอียดดังนี้:

> **Bangkok**
> Thailand
> 
> **32.5°** > ท้องฟ้าแจ่มใส
> 
> * **ความชื้น:** 65 %
> * **ความเร็วลม:** 12.5 km/h
> * **ปริมาณฝน:** 0.0 mm

*(กรณีเกิดข้อผิดพลาด เช่น พิมพ์ชื่อเมืองไม่มีจริง ระบบจะแสดงกล่องข้อความสีแดงแจ้งเตือนว่า "ไม่พบข้อมูลพิกัดของเมืองที่คุณระบุ")*

## 🔮 แผนการพัฒนาในอนาคต (Future Development Plans)
* **การพยากรณ์ล่วงหน้า:** เพิ่มการแสดงผลสภาพอากาศล่วงหน้า 3-7 วัน (Multi-day forecast)
* **การระบุตำแหน่งอัตโนมัติ (Geolocation):** เพิ่มปุ่มดึงพิกัดปัจจุบันของผู้ใช้เพื่อแสดงสภาพอากาศโดยไม่ต้องพิมพ์ชื่อเมือง
* **Dynamic Background:** ปรับเปลี่ยนสีพื้นหลังและไอคอนของเว็บแอปพลิเคชันให้สอดคล้องกับสภาพอากาศ ณ ขณะนั้น (เช่น พื้นหลังฝนตก, พื้นหลังกลางคืน)
* **รองรับหลายภาษา:** เพิ่มการรองรับการค้นหาและการแสดงผลสภาพอากาศเป็นภาษาไทย
